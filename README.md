## Featured Project Highlights

### 1. Computer-Graphics-Rendu

- Focus: shaders, lighting, projection, z-buffer, rasterization

### 2. Modelisation
Repository: https://github.com/aliwally/Modelisation

- Context: image modeling and geometric analysis
- Focus: SLIC/superpixels, binarization, skeleton, medial axis, Delaunay triangulation
- Stack: MATLAB

### 3. Segmentation-images
Repository: https://github.com/aliwally/Segmentation-images

- Context: image segmentation experiments
- Focus: k-means clustering and variants in C++
- Stack: C++, Makefile

### 4. seam_carving
Repository: https://github.com/aliwally/seam_carving

- Context: content-aware image resizing
- Focus: seam-carving algorithm and visual-preservation tradeoffs
- Stack: Python

### 5. Python_scripting_for_animation
Repository: https://github.com/aliwally/Python_scripting_for_animation

- Context: Maya pipeline scripting practice
- Focus: UI tool to auto-create clean animation project folder structures
- Stack: Python, Maya
